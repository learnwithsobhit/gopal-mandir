# SEO KPI Measurement Plan

## Primary KPI Targets (8-12 Weeks)

- Indexed pages: all sitemap pages indexed
- Query impressions: upward trend for:
  - `gopal mandir`
  - `gopal ji`
  - `laddu gopal`
  - `gopal ji mathura`
  - `mathura`
- Average position: improvement trend week-over-week
- CTR: increase on branded + intent terms

## Data Sources

- Google Search Console: queries, pages, CTR, average position
- Bing Webmaster Tools: query and crawl diagnostics
- Firebase Analytics / web analytics: landing page sessions and engagement

## Weekly Review Cadence

1. Export query report for last 7 days.
2. Compare positions and CTR to previous week.
3. Identify pages with high impressions but low CTR.
4. Update title/meta description of underperforming pages.
5. Expand bilingual content for missing long-tail queries.

## Monthly Iteration Actions

- Refresh homepage SEO section with new devotional/event context.
- Update static keyword pages with fresh copy and recent temple activities.
- Add 1 new long-tail keyword page if query data supports it.
- Re-submit updated URLs in Search Console.
