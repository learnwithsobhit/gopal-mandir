# Off-Page Authority Checklist

Target URL: `https://gopal-mandir-app.web.app/`

## Priority Submissions

- Google Business Profile (if official temple address exists)
- Facebook page profile URL
- Instagram profile URL
- YouTube channel profile URL
- WhatsApp channel/profile URL
- Devotional and temple directory listings

## Standard Listing Format

- Name: `Shri Gopal Mandir`
- Category: `Temple / Religious Organization`
- Website: `https://gopal-mandir-app.web.app/`
- Description (short): include "Gopal Ji", "Laddu Gopal", "Mathura bhakti"
- Description (long): bilingual summary of darshan, seva, aarti, and festivals

## Backlink Campaign (Month 1)

1. Publish 2 bilingual devotional articles on owned channels.
2. Share each article to 3 community groups/pages.
3. Request 5 relevant backlinks from devotional communities and local listings.
4. Add profile links from every social channel back to the website.

## Tracking Sheet Columns

- Date
- Platform
- Listing URL
- Status (`submitted`, `approved`, `rejected`)
- Anchor text used
- Follow-up date
- Traffic from source (weekly)
